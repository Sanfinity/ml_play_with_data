# ML Play with Data

A learning playground for mastering Python data analysis libraries: NumPy, Pandas, and Matplotlib.

## Contents

### Getting Started
- **00_jupyter_tutorial.ipynb** - Jupyter Notebook fundamentals
- **numpy_tutorial.ipynb** - NumPy essentials for numerical computing

### Pandas Basics
- **01_pandas_series.ipynb** - Working with Pandas Series
- **01_pandas_missing_data.ipynb** - Identifying and handling missing data
- **02_pandas_handling_missing_data.ipynb** - Missing data techniques
- **03_pandas_dataframes.ipynb** - DataFrames fundamentals
- **03_pandas_cleaning_null_values.ipynb** - Cleaning null values

### Data Visualization & Real Examples
- **04_matplotlib_visualizations.ipynb** - Data visualization techniques
- **05_pandas_csv_and_plotting.ipynb** - Reading CSV files and basic plotting
- **05_real_world_example.ipynb** - Real-world data analysis example

## Data Files

The `data/` folder contains sample datasets:
- `btc-market-price.csv` - Bitcoin price data
- `eth-price.csv` - Ethereum price data
